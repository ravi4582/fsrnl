===================================regex============================================================
^ -> start of string
$ - end of string
*- wildcard(0 or more), + -> 1 or more
. - any character
[0-9] - \d  - digit
\D not match digit
\s - match white space
\w - any word character
\W - other than words
| or a|b => a or b ! -> not
? = optional 0 or 1 time
\ => escape character \. \* \?

?= positive lookahead
